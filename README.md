 ﻿# ptCastisoft

## Descripción
Un proyecto sin componentes ni librerías externas, hecho con Next.js 16.1.4 y TailwindCSS. 
## Instalación

Clonar el repositorio:
```
git clone https://github.com/gregvp00/pt_nxt_tailw.git
cd .\pt_nxt_tailw\
```

Ejecutar el dev:
```
pnpm install
pnpm run dev
```
```
npm install
npm run dev
```

## Tecnologías
- Next.js 16.1.4
- TailwindCSS 4
- ReactDOM
